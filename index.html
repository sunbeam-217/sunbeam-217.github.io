手把手(hand in hand)教你升级ChatGPT4.0

一、注册WildCard

（一）进入WildCard官网https://wildcard.com/i/PAYGPT4

（二）按照提示输入手机号和验证码就ok了

注：WildCard 开卡可以选择 1 年或 2 年服务时间。1 年的年费是 11.99 美元，使用邀请码可以优惠 2 个刀，又可以喝杯奶茶喽。邀请码是PAYGPT4，复制（一）中链接粘贴即可。

（三）登录WildCard

二、ChatGPT Plus 升级

（一）打开浏览器新标签页，登录 chat.openai.com依次点击 登录 ，输入邮箱和密码（前提已有账号）

注：2024年4月1日OpenAI 宣布旗下对话机器人 ChatGPT，将允许用户直接使用而无需注册或登录，即进入chat.openai.com就可以和ChatGPT3.5进行聊天。在非注册或登录状态下使用 ChatGPT，聊天记录不会被保存，开启新的对话后历史对话将无法查询。与此同时，聊天记录分享、语音对话、自定义操作等功能也无法使用。OpenAI 表示，它已为无需注册即可访问 ChatGPT 的用户引入了额外的内容保护措施，例如阻止更广泛范围的类别中的提示和生成。除了 ChatGPT 的免费版本(即ChatGPT3.5 )，OpenAI 还将继续为个人、团队用户和企业提供付费版本，ChatGPT 4.0 、DallE3等仍需注册付费订阅。

（二）点击升级 Upgrade

登录自己的 OpenAI 帐户后，点击左下角的 Upgrade to Plus，在弹窗中选择 Upgrade plan。

选择套餐后跳转至支付界面

注：如果升级入口无法点击，那就访问这个网址，https://chat.openai.com/invite/accepted

（三）复制支付页面网址

在支付页面，复制现在的付款页网址 URL，这一段 pay.openai.com 开头的全部网址。请不要在支付页面填写任何信息，如果填写过，请返回上一步，重新进入这个页面再复制网址。

（四）使用 WildCard 一键升级 ChatGPT Plus

进入WildCard首页，选择ChatGPT Plus一键升级，将（三）中复制的URL粘贴到输入框内，选择一键升级即可！

如何获取OpenAI或GhatGPT账号

OpenAI在2024年4月1日愚人节这天，给广大网友送来了这样一份惊喜的礼物，不需要登录，就可以直接使用3.5了！

But！重点圈起来要考的，未登录使用chatGPT不能保存和ChatGPT的聊天记录！！！

此外，若想升级成尊贵的Plus用户或者升级为Team成员都是需要注册账号的。

如何拥有OpenAI账号呢？

  • 目前ChatGPT对于邮箱要求还是比较高的，国内邮箱（QQ、163等）基本都用不了，可以用Gmail、Proton等国外邮箱注册，如果没有的话就去注册一个；
  • 由于ChatGPT没有开放国区，所以需要一个可以魔法上网的工具，保证自己可以访问ChatGPT官网。注意现在一些魔法工具的IP被ChatGPT封了（尤其是免费的工具）
  • 国外手机号（2023年11月更新：现在不需要验证手机号就可以使用ChatGPT了，但是如果使用OpenAI API，则必须验证手机号）：在正常使用ChatGPT前我们必须要验证ChatGPT手机号，同样的国内的手机号无法验证，所以我们还需要准备一个国外手机号，这个可以用接码平台来实现，需要充值2美元（可使用多次）

ps: wildcard平台可免费注册3个账号，此外还提供国外手机号等服务，平台有更详细的介绍，也可加我微信备注购买ChatGPT账号。

GPT-4o最新官方简介及使用教程

OpenAI关于ChatGPT4o介绍的原文链接：
https://help.openai.com/en/articles/7102672-how-can-i-access-gpt-4-gpt-4-turbo-and-gpt-4o

什么是GPT4o ？

GPT-4o 是 OpenAI 的新旗舰型，可以实时进行音频、视觉和文本推理。GPT-4o 最初将以文本和视觉模型的形式出现在 ChatGPT 和 API 中（ChatGPT 将继续通过已有的语音模式功能支持语音）。具体来说，GPT-4o 将在 ChatGPT 免费版、Plus 版和 Team 版（企业版即将推出）以及 Chat Completions API、Assistants API 和 Batch API 中提供。

在 OpenAI API 中访问 GPT-4、GPT-4 Turbo 和 GPT-4o

与 GPT-4 Turbo 相比，GPT-4o 具有相同的高智能，但速度更快、成本更低、速率限制更高。具体来说

  • 定价：GPT-4o 比 GPT-4 Turbo 便宜 50%，输入代币为 5 美元/M，输出代币为 15 美元/M）。
  • 速率限制：GPT-4o 的速率限制比 GPT-4 Turbo 高 5 倍，最高可达每分钟 1000 万个代币。
  • 速度：GPT-4o 的速度是 GPT-4 Turbo 的 2 倍。
  • 视觉：在与视觉能力相关的评估中，GPT-4o 的视觉能力优于 GPT-4 Turbo。
  • 多语言：与 GPT-4 Turbo 相比，GPT-4o 更好地支持非英语语言。

GPT-4o 目前的上下文窗口为 128k，知识截止日期为 2023 年 10 月。

API接口的可用性

任何拥有 OpenAI API 账户的人都可以使用 GPT-4o，您可以在聊天完成 API、助手 API 和批处理 API 中使用该模型。该模型还支持函数调用和 JSON 模式。您也可以通过 Playground 开始使用。

成功支付 5 美元或以上（使用层级 1）后，您就可以通过 OpenAI API 访问 GPT-4、GPT-4 Turbo 和 GPT-4o 模型。

了解有关向 OpenAI 账户充值的更多信息，API 中当前的 GPT-4o 定价参阅 API 定价页面。

API接口的限制

请注意，ChatGPT 费率限制与 API 费率限制无关。您可以在此了解有关 API 费率限制的更多信息。

您可以在 API 平台的限制部分查看您的 API 费率限制。请耐心等待–我们预计会有大量需求，并将在未来几个月内扩大和优化我们的系统。

使用ChatGPT访问GPT-4o

ChatGPT免费用户

免费层用户将默认使用 GPT-4o，但使用 GPT-4o 发送信息的数量有限制，具体数量将根据当前使用情况和需求而定。不可用时，免费层用户将切换回 GPT-3.5。

免费用户使用高级工具发送信息也会受到限制，例如、数据分析、文件上传、浏览、发现和使用 GPTs、视觉

GPT-4o 具备先进的视觉功能，能更准确地理解您共享的图像。

ChatGPT Plus和Team用户

ChatGPT Plus 和 Team 用户可以访问 chatgpt.com 上的 GPT-4 和 GPT-4o，但使用上限更高。ChatGPT Plus 和 Team 用户可以从页面顶部的下拉菜单中选择 GPT-4o。自 2024 年 5 月 13 日起，Plus 用户在 GPT-4o 上每 3 小时最多可发送 80 条信息，在 GPT-4 上每 3 小时最多可发送 40 条信息。我们可能会在高峰时段降低限制，以保证 GPT-4 和 GPT-4o 可供最广泛的用户使用。ChatGPT Team 工作区用户的 GPT-4 和 GPT-4o 消息上限高于 ChatGPT Plus。请注意，未使用的消息不会累积（例如，如果您等待了 6 小时，那么在 GPT-4 上接下来的 3 小时内您将不会有 80 条消息可供使用）。

Chat GPT Enterprise

ChatGPT 企业计划专为满足大型企业的需求而设计，可无限高速访问 GPT-4o 和 GPT-4。ChatGPT 企业账户的新对话将默认使用 GPT-4o。ChatGPT 企业用户可以从页面顶部的下拉菜单中选择其他型号：

ChatGPT 企业版还提供企业级安全和隐私保护、更长的上下文窗口以处理更长的输入、无限制高速访问数据分析等高级工具、自定义选项等。
